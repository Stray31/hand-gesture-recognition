AVAILABLE GESTURES
- SWIPE LEFT = NEXT SLIDE (RIGHT ARROW KEY)
- SWIPE RIGHT = PREVIOUS SLIDE (LEFT ARROW KEY)
- RIGHT HAND PEACE SIGN = MOUSE CONTROL (TAKE NOTE DAPAT NAKA OPEN YUNG THUMB) 
- RIGHT HAND INDEX FINGER DOWN = LEFT CLICK (PWEDE I HOLD LEFT CLICK FOR DRAG)
- RIGHT HAND MIDDLE FINGER DOWN = RIGHT CLICK
- RIGHT HAND THUMB CLOSE = MOUSE STOPS MOVING
- RIGHT HAND CLOSED FIST + LEFT HAND INDEX AND MIDDLE FINGER RAISED = ALT TAB
- RIGHT HAND CLOSED FIST + LEFT HAND INDEX DOWN = ALT TAB MOVE SA LEFT
- RIGHT HAND CLOSED FIST + LEFT HAND MIDDLE DOWN = ALT TAB MOVE SA RIGHT
- PINCH = SCROLL UP OR DOWN
- 'OK' SIGN SA LEFT HAND = PiP MODE 
- PINKY UP RIGHT HAND = STOP/CONTINUE GESTURE RECOG
- BOTH PALMS OPEN FOR 1.5 SECONDS = ZOOM MODE 
- ZOOM MODE PALMS SPREAD OUTWARD = ZOOM IN 
- ZOOM MODE PALMS SPREAD INWARD = ZOOM OUT 
- BOTH FISTS CLOSED FOR 1.5 SECONDS = EXIT ZOOM MODE 
- LEFT PINKY UP = MOUSE LOCKED
- RIGHT HAND PINKY AND INDEX UP = VOLUME MODE 
- PINCH LEFT HAND = VOLUME DOWN
- PINCH LEFT HAND INDEX AND THUMB APART = VOLUME UP
